## 第一次实践流程

<Steps>
  1. **选择一个主题。** 可以是心理健康、职业、亲密关系、自我成长、情绪或视角。
  2. **选择一个素材。** 使用一篇文章、一集播客、一个视频、一个活动主题或一张情绪罗盘卡。
  3. **建立安全边界。** 分享应该是自愿的，先说明隐私和参与边界。
  4. **命名信号。** 问：我正在感受什么？是什么触发了它？这个感受可能在提醒我什么？
  5. **选择一个行动。** 以一周内可以完成的小行动结束。
</Steps>

<Aside type="tip" title="保持温和">
  Compassio 实践最好以邀请方式进行。不要强迫披露、诊断别人，或把反思变成表演。
</Aside>
